# Spring Boot 自定义 Banner

## 创建文件

首先在 Spring Boot 项目 `resources` 目录下新建 `banner.txt` 文本文件；

![创建](https://s2.loli.net/2023/06/13/TQRghDNEI3qpm1B.png)
